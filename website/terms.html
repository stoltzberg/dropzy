Условия использования

Последнее обновление: 1 января 2025 г.

1. Регистрация

Для использования сервиса необходимо создать аккаунт, указав действующий адрес электронной почты. Вы несёте ответственность за сохранность данных для входа в аккаунт. Один пользователь может иметь только один аккаунт.

2. Запрещённый контент

Запрещается загружать файлы, нарушающие законодательство, содержащие вредоносное программное обеспечение, материалы, защищённые авторским правом без разрешения правообладателя, а также любой контент, нарушающий права третьих лиц. Администрация оставляет за собой право удалять файлы без предупреждения.

3. Ответственность

Сервис предоставляется «как есть». Мы прилагаем все усилия для обеспечения доступности и сохранности данных, однако не гарантируем бесперебойную работу и не несём ответственности за потерю данных. Рекомендуем хранить резервные копии важных файлов.

4. Ограничения

Максимальный размер файла определяется тарифным планом. Срок хранения файлов ограничен и зависит от выбранного тарифа. Сервис оставляет за собой право вводить дополнительные ограничения для предотвращения злоупотреблений.

5. Удаление аккаунта

Вы можете удалить аккаунт в любое время через настройки профиля. При удалении аккаунта все загруженные файлы будут безвозвратно удалены. Администрация может заблокировать или удалить аккаунт при нарушении настоящих условий.
