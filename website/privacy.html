Политика конфиденциальности

Последнее обновление: 1 января 2025 г.

1. Сбор данных

Мы собираем минимальный объём информации, необходимый для работы сервиса: адрес электронной почты при регистрации, IP-адрес для обеспечения безопасности, а также технические данные о браузере и устройстве для улучшения качества обслуживания.

2. Использование данных

Собранные данные используются исключительно для предоставления услуг файлообмена, обеспечения безопасности аккаунтов, отправки уведомлений о статусе файлов и улучшения работы платформы. Мы не передаём ваши данные третьим лицам без вашего согласия.

3. Хранение данных

Загруженные файлы хранятся в зашифрованном виде на серверах, расположенных в дата-центрах Европы. Файлы автоматически удаляются по истечении срока хранения, определённого вашим тарифным планом. Персональные данные хранятся в течение всего срока действия аккаунта.

4. Права пользователя

Вы имеете право запросить копию своих персональных данных, потребовать их удаления, отозвать согласие на обработку, а также подать жалобу в контролирующий орган. Для реализации этих прав обратитесь к нам через форму обратной связи.

5. Контакты

По вопросам, связанным с конфиденциальностью, обращайтесь через форму обратной связи.
